# gitPush.sh

```bat
#!/bin/bash

comment="Saved on $(date '+%Y-%m-%d %H:%M:%S')"
if [ -n "$1" ]; then
  username="$1"
  comment="$comment by $username"
fi

echo "$comment"

git checkout development
git add .
git commit -m "$comment"
git push origin

```
